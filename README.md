# FWAS: Fidelity-Weighted Adaptive Stringency

Implementation of the FWAS decision-theoretic framework for context-sensitive 
significance thresholds in biomedical research.

## Contents
- `FWAS_v1.0.0.ipynb` — fully executable notebook reproducing all results in 
  Tables 1–5 from fixed random seed 42

## Requirements
Python ≥ 3.8, NumPy ≥ 1.21, SciPy ≥ 1.7

## Usage
Run directly in Google Colab — no installation required.

## Reference
Murtaza, M.A. Fidelity-Weighted Adaptive Stringency: A Decision-Theoretic 
Framework for Context-Sensitive Significance Thresholds in Biomedical Research. 
Biostatistics (under review).
